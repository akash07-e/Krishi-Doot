# 🌱 KrishiDoot

KrishiDoot is an AI-powered Smart Agriculture Platform designed to help farmers make smarter farming decisions using Artificial Intelligence, Machine Learning, and IoT-based solutions.

The platform provides crop recommendations, fertilizer guidance, water optimization, weather-based insights, and a digital marketplace to support sustainable and modern farming.

---

# 🚀 Live Features

✅ Crop Recommendation using Soil + Weather Inputs  
✅ Fertilizer Advisor for Multiple Crops  
✅ AI-Based Smart Farming Assistance  
✅ Water Saving Planner  
✅ Farmer Marketplace & Community Support  
✅ User Authentication (Firebase Login / Signup)  
✅ Prediction History Stored in Firestore  
✅ Real-Time Dashboard & Analytics  
✅ Responsive Modern UI

# 🌾 Problem Statement

Traditional farming methods face several challenges:

- Water wastage due to inefficient irrigation
- Incorrect fertilizer usage affecting soil health
- Crop losses caused by unpredictable weather
- Lack of localized farming guidance
- Limited access to affordable smart farming tools

KrishiDoot solves these problems using AI-powered recommendations and smart agricultural insights.

# 💡 Core Modules

## 🌱 Crop Recommendation System
Suggests suitable crops based on:
- Soil Nutrients (NPK)
- Temperature
- Humidity
- Rainfall
- pH Value

## 🧪 Fertilizer Advisor
Provides fertilizer suggestions to improve:
- Soil fertility
- Crop productivity
- Sustainable farming

## 💧 Water Saving Planner
Helps farmers optimize irrigation and reduce water wastage.

## 🤖 Krishi AI Assistant
An intelligent farming assistant for:
- Crop guidance
- Fertilizer recommendations
- Weather insights
- Farming support

## 👨‍🌾 Farmer Marketplace
A digital marketplace where farmers can:
- Buy/Sell products
- Connect with others
- Share farming knowledge
