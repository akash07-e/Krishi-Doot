# 🌱 KrishiDoot

### AI Powered Smart Farming Intelligence Platform

**Smart Farming • Higher Income • Lower Risk**

KrishiDoot is an AI-powered smart agriculture platform designed to help farmers make intelligent and data-driven farming decisions using Machine Learning, weather analysis, soil insights, and market intelligence.

The platform helps farmers improve productivity, reduce farming risks, optimize resources, and maximize profits through smart recommendations and predictive analytics.

---

# 🚀 Key Features

## 🌾 Smart Crop Planner

AI-powered crop recommendations based on:

* Soil composition
* Weather conditions
* Historical crop patterns
* Profitability trends

## 💧 Water Saving AI

Smart irrigation guidance using:

* Soil moisture analysis
* Weather conditions
* Crop water requirements

Helps farmers reduce unnecessary water usage and improve irrigation efficiency.

## 🧪 Fertilizer Intelligence

Provides accurate fertilizer recommendations using:

* Soil NPK values
* Soil health analysis
* Crop nutrient requirements

## 📈 Mandi Price Predictor

Helps farmers maximize profits through:

* Nearby mandi price analysis
* Best selling time recommendations
* Market trend predictions

## 🎙 Voice Assistant

Supports Hindi and regional languages for easier farmer interaction.

## 👨‍🌾 Farmer Community

A collaborative platform where farmers can:

* Share farming knowledge
* Discuss challenges
* Learn best practices
* Build local support networks

---

# 🌍 Problems Addressed

Traditional farming faces several major challenges:

* Water wastage due to inefficient irrigation
* Incorrect fertilizer usage damaging soil quality
* Unpredictable weather conditions
* Poor crop planning decisions
* Lack of market price awareness
* Dependence on middlemen reducing profits
* Limited access to personalized farming guidance

KrishiDoot solves these problems using AI-driven recommendations and smart farming intelligence.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/AnshSinghRathoree/KrishiDoot.git
```

## Move to Project Folder

```bash
cd KrishiDoot
```

---

# ▶️ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

# ▶️ Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

---

# 🔐 Firebase Configuration

Create a `.env` file inside frontend directory and add:

```env
VITE_FIREBASE_API_KEY=YOUR_API_KEY
VITE_FIREBASE_AUTH_DOMAIN=YOUR_DOMAIN
VITE_FIREBASE_PROJECT_ID=YOUR_PROJECT_ID
```
